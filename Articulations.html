Articulations Explanation

Regular - Samples without effect (aka Vanilla / Normal).
Cool! - Velocity Switch and chorus effect.
Live! - Stereo and reverb effect.
Sweet! - Vibrato'd sample.
Natural! - High Quality Samples. "Clavinova" and DGX-650 Exclusive.

 OrganFlutes - "setBfree" like UI, with default parameters, and can be emulated.
This goes to either "Regular" or "Cool!" categories for samplers.
VRM - Virtual Resonance Model. "Clavinova CVP-700+ and CSP-150+" Exclusive.
This can be emulated with external resonance (Like Salamander Piano) but it doesn't perfect. Thus, goes to "Natural!" category for samplers.

MegaVoice - Different articulations with different velocity range and key ranges.
Unpractical in software samplers (e.g. SoundFonts), since MegaVoices uses the fixed volume for each
velocity (Velocity range is nullified if you set fixed velocity in SoundFonts).

 S.Art! / S.Art2! - Super Articulations. Affacts polymode, instrument layer each time played, etc.
Therefore, It's undoable in any software samplers (e.g. TX16Wx, SoundFonts, etc.), nor emulate them.
I think SA2 is patented.

Live! Series Percussion - Stereo Percussion, Reverb FX Included.
Revo! Series Percussion - "Genos" Exclusive. Round Robin (Like SC-8850's Standard L/R Kit). Only doable in Hydrogen.
