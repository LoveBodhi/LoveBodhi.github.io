Record from acoustic instruments
Compare with sample-based synthesizer / Arranger Keyboards, then use libre or self-recorded samples to make (Don't sample directly)
 Record from a non-free or commercial sources (e.g. VSTi's, SFXs. You're infringed the law if you do this!!!)
 Record directly from a sample-based digital synthesizers / Romplers / Arranger Keyboards (Samples are copyrighted by both buyers and producers therefore can't licensed under a free license!). You just buy and play them, no other options.
Link-up Video Game / Mobile materials (Copyrighted by devs. This will illegal because you're infringed the copyrights!)
Link-up Crisis GeneralMIDI materials (These are copyrighted/protected and archived!! I have the right to report to correspond site admin if it's came from CGM!!)
Use libre "PCM" Samples for synthesized tones
Record your own "PCM" Samples for synthesized tones		 Record from linear synthesizers ("PCM" samples are copyrighted by both buyers and producers). It's not recommended for sample source. Super LA (D-70) is full PCM. Thus, NEVER use that for sample source!!!! Use non-free/non-libre materials on open source materials (Such as: CC Sampling+, "Philharmonia" restriction [This means prohibit for making sample lib! Now down], etc.)
Create derivatives of "All Rights Reserved" (exclude BSD and MIT) materials
"Creditless" collections (This should be "Grey Area". We can't found the source easily)
Re-license restrictive (Non-free/non-libre) materials under a free license (They're no license knowledge). You have to report to Sample Lib creator, Polyphone Website Admin (Davy), etc. Must choose "Licensing Grey Area" if it's not CC-BY-NC licensed.
Ignore license field on Musical Artifacts, cause License Mis-match (Contact LFZ if this exists!)
Use CC older than 3.0 (CC-BY 3.0+ is the mininum of GPL Compatible License, along with CC-BY-SA 4.0)
Use permissive / copyleft licensed materials
(Even BSD license have "All Rights Reserved", This is still freely-licensed)
Release to Public Domain (CC0. Don't use WTFPL. I've contacted LFZ for add CC0)
Combine all GPL-compatible works (Not Copyleft works or GPL works by itself). The final work must be licensed on a fully compatible licenses.
Attach License File on sf2 (This is called License Override)		 Re-license copylefted materials
Re-license freedom materials under non-free license

GPL Works specific: Use CC-BY-SA 3 (CC-BY 3 and 4 are GPLV3 compatible)
Use lossless audio formats like FLAC, WAV, etc. Use lossy audio formats (1-way)
Use SF2 or SFZ (Direct Samples) Use compressed formats, such as SFARK, SFPACK and SFOGG, etc. The specific player must be decoded such as Falcosoft (CC-BY-NC-SA), SFPack, SFARK and BASSMIDI. All are NON-COMMERCIAL USE ONLY!!!!!!!!!!!!! (Unless lossy sf3 and FLAC-compressed sf4 [Pure GPL V2]) The latter 2 are donation for commercial usage.
Use GIG (Current only LMMS and non-free LinuxSampler support)
 Use Virtual MIDI Cables on normal purposes other than Linux (All non-commercial use only on Windows and non-free for Mac (Even commerical use is permitted). All freely-licensed Virtual MIDI Cables are ONLY available in Linux!!!). Buy a MIDI Keyboard instead.
Record from a sample lib (If it's libre, use directly-process method instead)
Use NKI / EXS (Shareware only format). It can be extracted if the file is non-monolithic and Libre Samples (Such as CdM Gamelan, AL V2)
Record stereo samples for acoustic/electric instruments and Synth Pads
Record mono samples for synth instruments
Record all notes as possible. (This will waste spaces. Also can: 3-4 per octave for acoustic, 2-4 per octave for synth)
Record different articulations
Record different velocity layers, or use Filter Cutoff on velocity layers
Record single notes (Oneshot)
Mix different samples ([Subtractive or FM] + ["Libre + Realistic" or self-recorded acoustic PCM]), This is "Linear Arithmetic", one of Roland's technology
Use LMMS to control a libre / FS Synthesizer VST
If Linear Synthesizers are an issue, use "Surge" to recreate them (Put Libre/Self-recorded + Acoustic PCM Samples on Wavetable. Use "DUAL" scene) 		Record via amplifier (Later can be processed via Audacity/LMMS).
Record samples that less than <44.1khz, 16bit
Skip too many notes
Use Synister (Ailas Issue)
Add DSP FX internally (This is sample lib exclusive. Later can be processed via Audacity, except of Chorus, that require LMMS)		 Use Guitar Rig (Paid Material!!! Guitarix is not available on Windows or Mac! Use LMMS or Audacity if you're using Windows or Mac)
Record from a non-libre and commercial Synthesizer VSTi's (E.g. Most vst4free Synth VSTi's)
